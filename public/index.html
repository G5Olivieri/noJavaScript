Por que não usar JavaScript no backend?

Node

Desde o surgimento do Node.js que levou o motor V8 do Chrome para o servidor, o JavaScript voltou ao patamar das linguagens mais utilizadas, e está em constante processo de popularidade, ou seja, a cada dia que passa mais programadores aderem o uso do JavaScript no seu dia a dia. Essa popularidade da linguagem é excelente, porque com grande foco nessa linguagem, e muitos profissionais envolvidos, todos os dias surgem novas opções de recursos, o que torna a linguagem cada vez mais amigável de trabalhar.
